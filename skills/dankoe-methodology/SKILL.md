## 六大模块体系

### 模块① 观点 > 垂直领域（Perspective > Niche）

**核心命题**: 你不是内容工厂，你是看待世界的透镜。人们不追随字典，他们追随人。

**关键工具**:
- **Trust Matrix 信任矩阵**: 可信度 + 真实性 + 权威性
- **定位公式**: 「I help [X audience] achieve [Y outcome] by [Z method]」
- **使命驱动利基策略**: 业务成为人生工作的载体
- **Content Pillars 法**: 3-5 个核心主题服务于核心定位
- **Idea Synthesis**: 差异化 = 独特经历 × 知识组合 × 表达方式
- **Brand Story Engineering**: Forge→Breakthrough→Transformation
- **现实扭曲力场**: 知道自己想要什么且不妥协

**使用场景**: 选赛道 / 个人品牌定位 / 内容方向不明确

### 模块② 策展思维（Curation > Creation）

**核心命题**: AI 时代任何人都能创作，稀缺的是品味和辨别力。策展 > 创作。

**关键工具**:

**阅读方法论**
- **狩猎式阅读**: 为找到改变人生的一个想法而读，而非读完一本书
- **5 分钟阅读习惯**: 每天只读一个 impactful 想法
- **Commonplace 笔记法**: 捕获最突出的想法→停下思考→转译
- **深挖兔子洞**: 停止主流内容，最好的想法在鲜为人知的书/播客中

**品味养成**
- **品味 = 觉察思维矩阵的能力**: 对自身认知局限的觉察
- **怀疑论者心态**: 怀疑他人信念，也怀疑自己的怀疑
- **原创思维三步骤**: 同中求异 + 读少数人的书 + 跨学科连接

**信息筛选/噪声过滤**
- **99%信息是干扰**: 压缩心智到少数重要事情
- **框架过滤**: 愿景/元目标天然屏蔽不相关信息
- **编程或被编程**: 策展你的数字环境
- **廉价 vs 昂贵多巴胺**: 增加信号噪声比

**输入系统**
- **沉浸式学习**: 看完长视频凭记忆写 400 字总结
- **建造者心态**: 始终有项目在构建，项目引导知识
- **深度学习六步法**: 设定目标→安排时间块→笔记→构建→教学→转化

**Content Stack**
- 底层=原始知识 → 中间层=合成框架 → 上层=发布内容
- AI 用于发现已有知识的连接

**使用场景**: 提升内容质量 / 信息筛选标准 / 知识管理

### 模块③ 2 小时内容生态

**核心命题**: 每天 2-4 小时纯粹专注。短内容做研究展示，长内容做深度表达。

**关键工具**:
- **内容飞轮**: Read → Write → Repurpose → Productize → Distribute
- **Hub & Spoke**: Newsletter 是 Hub，社交平台是 Spokes
- **想法验证法**: X 推文测试→验证后扩展长内容
- **3 层 Prompt**: Swipe File → Structure → Transformation
- **每周聚焦**: 「One core idea, one week, one deep output」
- **叙事写作框架**: Hook → Tension → Resolution → CTA
- **AI 思考对齐法**: AI 提供骨架，人类提供血肉
- **平台差异化**: X=想法实验室 / Newsletter=深度工作间 / YouTube=叙事舞台

**使用场景**: 安排创作时间 / 跨平台分发 / AI 辅助写作

### 模块④ 一人公司系统

**核心命题**: 三大财富决策：荒谬时薪、力量集中、不留退路。

**关键工具**:
- **三大致富决策**: $5,000/h 心态 × 力量集中 × 不给退路
- **Value-Productization Protocol**: 知识→可扩展数字产品
- **Creator→Educator→Founder**: 渐进式转型
- **最小风险创业路径**: 卖服务→建受众→推产品→扩展
- **免费→付费漏斗**: Free → $19 → $200-500 → $500+
- **产品阶梯**: 引流品→入门→核心→进阶/工具
- **1000 真粉丝**: 不到 1000 人每月 $5-10k

**使用场景**: 内容变现 / 数字产品设计 / 一人企业起步

### 模块⑤ 技能习得（Skill Acquisition）

**核心命题**: 技术堆叠法。从目的出发，拆解核心技术，2 周学到足以产生结果。

**关键工具**:

**技术堆叠**
- **Technique Stacking**: 每天一个微技术点，90天=90个互连点
- **T型技能堆叠**: 深度(写作/沟通) + 广度(设计/编程/营销/AI)
- **抗衰技能栈**: 写作→设计→编程→销售→AI
- **常青 vs 趋势技能**: 常青做深，趋势叠加

**写作——元技能**
- **写作三层级**: 写给自己→写给他人→写为系统
- **写作反馈环**: 每天500字→AI分析→修改→发布→迭代
- **写长文=强迫跨域合成**

**刻意思考/深入练习**
- **逆向学习**: 拿到成品，不靠教程复现
- **刻意练习 ≠ 重复**: 找弱点→针对性练习
- **好奇心加速器**: 好奇驱动学习速度快10倍

**心理障碍处理**
- **冒名综合征=成长信号**: 作品积累量取代自我怀疑
- **高自主权心态**: 不需许可就开始学习
- **注意力肌肉**: 每天 30min 深度工作，逐渐延长

**反馈机制**
- **公开展示学习**: 社交媒体接受检验
- **先行动后学习(Just-in-Time)**
- **通过教来学**: 公开承诺教授创造责任感

**技能→产品化桥梁**
- **Build-Teach-Earn**: 建项目→教学→打包付费
- **写作是营销，产品是框架**

**使用场景**: 学新技能 / 快速上手 / 技能变现 / 学习卡住

### 模块⑥ 身份工程（Identity Engineering）

**核心命题**: 纪律不是靠意志力，是身份的副产品。

**关键工具**:
- **身份工程四步骤**: 识别身份→策略性失调→环境工程→自我实验
- **一夜转变法**: 撕掉创可贴，跳入新身份
- **整体僧侣模式**: 1项目+1书+1冥想+1训练，消失3-6月
- **12 个大胆决定**: 错误决定 > 不做决定
- **系统 > 目标**: Goals have an end, systems are forever
- **谬误目标重写大脑**
- **生活游戏化**: 目标=决策镜头

**使用场景**: 培养自律 / 改变习惯 / 人生重大决定
