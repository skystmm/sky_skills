## 六大模块

### 模块① 观点 > 垂直领域
**核心命题**: 你不是拍照片的，你是用影像创造视角的人。

**常见坑**：
- ❌ "我是风光/人像/街拍摄影师"（技能标签）
- ✅ "我帮助城市疲惫的人**通过镜头重新发现日常之美**"（世界观的表达）

**定位公式**:
> 「我帮助 [X 受众] 实现 [Y 结果] 通过 [Z 方法]」

示例：
- "我帮助**手机党**拍出**让朋友问『这是相机拍的吗』**的照片，通过**极简构图+手机后期三步法**"
- "我帮助**摄影爱好者**建立**自己的影调风格**，通过**大师作品拆解+每周一个视觉实验**"

**Brand Story Engineering** (自我品牌故事):
- **Forge**: 哪里开始的？拍什么？为什么拿起相机？
- **Breakthrough**: 哪次拍摄/哪张照片让你"明白了"？
- **Transformation**: 现在你是谁？你教/帮别人什么？

**使用场景**: 定位模糊、不知道怎么自我介绍、内容没方向

### 模块② 策展思维 — 策展 > 拍摄
**核心命题**: AI 能生成任何照片。摄影师的护城河不是快门，是品味——决定拍什么、怎么编、为什么按快门。

**输入策展体系**:
- **大师输入源**: 何藩 / Ansel Adams / 杉本博司 / 薇薇安·迈尔 / Alex Webb
- **视觉营养**: 电影 / 绘画 / 建筑 / 平面设计（不要只看摄影）
- **阅读**: 《论摄影》(Sontag)、《明室》(Barthes)、《风景的诱惑》
- **摄影论坛输入**: 色影无忌 / 蜂鸟论坛 / DPReview
- **当前 KOL 输入**: 小红书摄影区 KOL + 比赛获奖者社媒（ins/X/小红书/抖音）

**品味建设方法**:
- **对比分析法**: 同一场景/主题，找 3 位不同风格摄影师的作品对比
- **回看反刍法**: 每月回看你上个月的作品，选最差的 3 张分析为什么
- **跨界汲取**: 从电影镜头语言、平面广告、建筑空间学习中构图逻辑

**噪声过滤**:
- **99% 摄影内容是干扰**: 每天产生无穷量"参数帖""教程帖"，大部分是重复信息
- **聚焦你的视觉项目**（比如"拍100个雨夜"），所有输入只服务于这个项目
- **不追器材**：新相机评测→噪声。**只用现有器材拍出好照片的方法**→信号

**使用场景**: 提升审美/风格、建立价值判断标准

### 模块③ 2 小时内容生态
**核心命题**: 每天 2 小时专注创作。短内容做展示（吸粉），长内容做深度（建立信任）。

**内容飞轮**:
```
拍摄现场 → 选片修图 → 小红书发9图+文案 → 视频号发拍摄vlog → 
写一篇深度拍摄笔记(小红书长文/公众号) → 打包后期配方为预设产品
```

**平台差异化**:

| 平台 | 角色 | 内容形态 |
|------|------|----------|
| 📱 小红书 | 想法实验室+作品展示 | 9图对比+拍摄思路+参数+修图教程 |
| 🎬 视频号 | 叙事舞台 | 拍摄vlog、后期过程、作品集影片 |
| 📧 公众号/专栏 | 深度工作间 | 一篇拍摄的完整心路历程与技术拆解 |
| 📸 500px/图虫 | 作品存档 | 全尺寸作品，链接回社媒 |
| 🌍 Instagram | 国际作品输出 | 视觉为主，文案辅助 |

**每周聚焦原则**:
每周只探索一个视觉主题或技法。比如：
- 周1:"蓝色时刻"的光质分析
- 周2:在不同场景拍5张蓝色时刻
- 周3:后期统一调色方案
- 周4:小红书发对比+思路
- 周5:视频号发拍摄过程
- 本周所有内容、拍摄、后期都围这个主题转

**叙事写作框架(摄影版)**:
- **Hook**: "这张片我在零下10度等了2个小时"
- **Tension**: "按快门的瞬间心里是没底的——LCD里一片黑"
- **Resolution**: "回家拉曲线才发现暗部藏了一个故事"
- **CTA**: "你的手机/相机也能拍，只需这一个技巧..."

**AI 对齐法**:
AI 帮你整理文案结构、生成标题建议、提炼拍摄笔记要点。但视觉审美判断、时机、情感——这些 AI 不能替代。

**使用场景**: 内容创作规划、多平台分发、拍摄-发布流程

### 模块④ 一人公司系统
**核心命题**: 三决定：把时薪定到荒谬的高度、力量集中、不给退路。

**产品阶梯**:

| 层级 | 产品类型 | 示例 | 定价 | 目的 |
|------|----------|------|------|------|
| 🆓 免费引流 | 内容 / 教程 | 小红书笔记 | $0 | 证明实力+建信任 |
| 💰 引流品 | 预设包/配方 | "城市夜景预设5套" | ¥29-99 | 低门槛收入 |
| 💵 核心品 | 课程/训练营 | "30天手机摄影审美课" | ¥299-999 | 深度教学+高利润 |
| 💎 高价品 | 1对1 Portfolio Review | 作品点评+风格定位 | ¥999+ | 限量服务 |

**最小风险创业路径** (6-12个月):
1. **月1-2**: 发内容(小红书+视频号)，每周3-5条，建受众
2. **月3-4**: 打包预设包，卖 ¥49。用收入验证方向
3. **月5-6**: 基于预设包的话题热点，开第一期训练营
4. **月7-12**: 升级系统课程 + 限量1对1服务 + 摄影周边

**1000 真粉丝摄影版**:
不需要 10 万粉丝。1000 个真心喜欢你的影调、为你预设/课程付费的人 = 月入 ¥5-15k。

**关键引用**:
- "A personal brand is not a business model, it's a trust mechanism."
- "If you don't sell a product, you will be forced to sell your labor for someone else."

**使用场景**: 变现规划、产品设计、从爱好者到创业

### 模块⑤ 技能习得
**核心命题**: 技术堆叠——技法完备后，用新工具/新媒介持续扩展能力栈。

**摄影人的技能堆叠**:

| 核心技能 | 辅助技能 | 为什么 |
|----------|----------|--------|
| 摄影(曝光/构图/用光/后期) | ★ 核心 | 你已经够了 |
| 写作(叙事/文案/表达) | ★ 高需 | 叙事让作品溢价 |
| 视频拍摄+剪辑 | ★ 高需 | 小红书视频+视频号变现 |
| AI 后期工具 | ★ 高需 | 降噪/超分/局部编辑/AI调色，加速5倍 |
| 基础设计 | ★ 中等 | 你的主页/预设UI需要审美匹配 |
| 营销+销售 | ★ 中等 | 没人知道你的预设就不会买 |

**逆向学习法(摄影版)**:
找到一张你喜欢的照片→不看教程自己去复现→被迫分析光照/机位/影调/色彩→学到的比看10个教程都快。

**好奇心驱动**:
技巧已经够用了→对什么视觉主题真正好奇？(蓝晒法/红外摄影/胶片翻拍/光绘)→围绕好奇心展开学习。

**即时学习**:
刚出一个新 AI 修图工具→不读说明书→直接上手用它精修一张废片→遇到不懂的参数再查。

**通过教来学**:
学了新技巧→马上发一条小红书教别人→被迫把模糊理解变成清晰表达→读者的提问是最真实的反馈循环。

**使用场景**: 学新技法、跟新工具、突破瓶颈期

### 模块⑥ 身份工程
**核心命题**: 纪律不是靠意志力，是身份的副产品。"我是谁"决定了我做什么。

**身份转变路径**:
1. **"拍着玩的"** → 2. **"我是摄影师"** → 3. **"我是视觉创作者"** → 4. **"我是摄影教育者/创作者"

**身份工程四步骤(摄影版)**:
1. **识别当前身份**: 你现在的言行围绕什么身份？("我也就是拍拍玩")
2. **策略性失调**: 把理想身份的社交媒体主页贴在眼前，天天看差距
3. **环境工程**: 取关所有"器材党"账号，关注在教你方法论的摄影师；相机永远带在身边
4. **自我实验**: 花 30 天每天拍一张同一个主题——30天后你就不再是"拍着玩"的人

**一夜转变法**:
明天发一条朋友圈/小红书，不要说"我是摄影爱好者"——说 "我是一个用镜头记录城市呼吸的人"。

**系统 > 目标**:
- ❌ 目标：今年小红书做到1万粉
- ✅ 系统：每周拍1组作品→写1篇深度笔记→教一个人你的方法→迭代你的预设

**僧侣模式(摄影版)**:
消失一个月，只做4件事：每天拍一个项目 + 每天读一本画册/摄影书 + 每天冥想(观察练习) + 每天一次身体训练(体能支持拍摄)。30天后你的视觉会完全不一样。

**使用场景**: 自我定位、习惯养成、瓶颈期突破

## 底层世界观

- **摄影不是记录，是翻译**: 把三维世界翻译到二维平面的过程——你的所有选择(焦段/光圈/快门/后期)都是翻译的语言
- **视觉是无限游戏**: 没有"学完了"的摄影，每一个新项目都是一次重新开始的游戏
- **技法够用，风格不够**: 当所有摄影师参数都差不多时，你的视角和坚持就是你的护城河
- **创作义务**: 每个独特的视角都有义务被分享——你没见过的世界，别人也还没见过

## 自迭代架构

### 知识引擎层 (自动 + 手动)
Skill 内置一个每周自更新的知识基线：

| 基线 | 更新方式 | 内容 |
|------|----------|------|
| photographers.json | 每周抓取 | 摄影 KOL 库（小红书/ins/X/抖音 KOL + 比赛获奖者） |
| techniques.json | 每周抓取 | 新兴摄影技术/后期技巧/教程 |
| tools.json | 每周抓取 | 新器材/软件/AI 工具评测摘要 |
| trends.json | 每周抓取 | 摄影圈热点话题/比赛/展览动态 |

### 使用迭代钩子
每次调用本 skill 时:
1. 检查 `references/` 下的最新数据
2. 如果上次更新超过 7 天 → 标记 "建议手动 ingest"
3. 引用到的数据标记 `source_date`
4. 旧引用(>30天) 标记 `[可能已过时]`

### 手动触发
```
python3 scripts/ingest.py --force
```
忽略上次更新时间，强制全量刷新所有基线。
